# Epic GRP Frontend

Stack:
- Vite + React + TypeScript
- React Router
- TanStack Query
- Axios (com Authorization)

Scripts:
- `npm run dev` — dev server (VITE_API_URL para apontar para o backend)
- `npm run build` — build
- `npm run preview` — preview do build

Env:
- `VITE_API_URL` (ex.: http://localhost:8000)
